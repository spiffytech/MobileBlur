MobileBlur is a phone- and tablet-friendly interface to the Newsblur RSS reader. It's designed to be easy to use on devices with small screens and slow processors.

If you like MobileBlur, please let me know with the feedback link below! If you want something added/improved, please let me know and I'll consider it.

MobileBlur takes your security seriously. MobileBlur relies on browser cookies to access your Newsblur account while you're navigating MobileBlur. It does not store anything that could be used to access your Newsblur account at any other time.

The MobileBlur source code is available here if you'd like it.
